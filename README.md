# LetramentoEstat-stico
Este repositório faz parte de um projeto de extensão da Universidade Federal Fluminense (UFF) que transforma o ensino de Estatística em experiências lúdicas, práticas e colaborativas. Aqui compartilhamos atividades, jogos e ideias que aproximam a Estatística do dia a dia e fortalecem o letramento estatístico em diferentes etapas da educação básica.
